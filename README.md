# AI 饮食智能助手
基于大模型 + MCP 协议的智能饮食助手

## 项目功能
- 自然语言聊天
- 对话记忆功能
- 自动记录饮食
- 饮食热量查询
- 家常菜推荐
- 支持 MCP 标准工具调用

## 技术栈
- Python + Streamlit
- 大模型自主思考
- MCP 协议工具
- 可在 MCP Inspector 中测试工具

## 交付文件
1. **app.py** - AI 智能体主程序
2. **mcp_diet_tool.py** - MCP 标准工具代码

## 核心特点
- 纯大模型自由思考
- 支持两个 MCP 工具：记录饮食、自由问答
- 界面简洁、记忆持久、运行稳定
